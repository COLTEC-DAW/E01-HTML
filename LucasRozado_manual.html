Manual de Programação em C

Introdução

C é uma linguagem de programação compilada de propósito geral, estruturada, imperativa, procedural, padronizada pela Organização Internacional para Padronização (ISO), criada em 1972 por Dennis Ritchie na empresa AT&T Bell Labs para desenvolvimento do sistema operacional Unix (originalmente escrito em Assembly).

C é uma das linguagens de programação mais populares e existem poucas arquiteturas para as quais não existem compiladores para C. C tem influenciado muitas outras linguagens de programação (por exemplo, a linguagem Java), mais notavelmente C++, que originalmente começou como uma extensão para C.

Estruturas Condicionais

Operador Comparativo Retorna verdadeiro se...
A == B A é igual a B
A != B A é diferente de B
A > B A é maior que B
A < B A é menor que B
A >= B A é maior ou igual a B
A <= B A é menor ou igual a B
!A A é falso
A && B A e B são ambos verdadeiros
A || B A ou B ou ambos são verdadeiros

else if (condição)
{
...
}
 Realiza "..." se a condição do "if" o qual segue for falsa
e a própria condição for verdadeira.		 
else
{
...
}
 Realiza "..." se a condição do "if" o qual segue for falsa. 
switch (X)
{
  case A:
    ...
    break;

  case B:
    ...
    break;

  default:
    ...
}
 		Compara "X" com cada caso (nesse exemplo, "A" e "B")

Exemplo:

{
    int inteiro = 5;

    if (inteiro != 8)
    {
        inteiro = 8;
    }

}

Estruturas de Repetição

Função Comportamento

}
Testa a condição e, se for verdadeira,

do

 Realiza "...", testa a condição e,

for (X; condição; Y) 
{
 ... 
}
 Executa "X" uma única vez, testa a condição e,
se for verdadeira, realiza "...", realiza "Y"

Exemplo:

int main()
{
    int inteiro;
    for (inteiro = 0; inteiro <= 10; inteiro++)
    {
        printf("%d", inteiro);
    } 

    return 0;
}

Como executar um programa em C

  1. Localize ou salve o arquivo .c no seu computador, de preferência em um diretório de fácil acesso.

  2. Abra o terminal (Linux) ou o command prompt (Windows).
    • Para executar um programa pelo terminal ou pelo command prompt, é necessário ter um compilador gcc instalado. Para checar a existência de um em sua máquina, execute o comando gcc -v. Se o comando não for executado, certifique-se de instalar o compilador gcc.

  3. Navegue até o diretório onde o arquivo .c está localizado utilizando o comando cd. Caso seja necessário, faça uso do comando dir para listar os diretórios e arquivos da pasta atual.

  4. Já no diretório do arquivo .c, execute o comando gcc nomeDoArquivo.c -o nomeParaOExecutavel. Isso irá compilar o arquivo e prepará-lo para execução, criando um executável com o nome dado como argumento para o comando.

  5. Agora basta executar o arquivo "nomeParaOExecutavel" com ./nomeParaOExecutavel (em Linux) ou simplesmente nomeParaOExecutavel (em Windows).
