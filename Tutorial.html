Introdução

A linguagem C, que em breve vão aprender, é uma linguagem de programaçaõ estrutural (entenderão esse termo mais posteriormente no curso) e multi-plataforma (praticamente para todas as plataformas foi implementado pelo menos um compilador C).

Algumas caracterísicas de programas em C são:

  • Um programa em C é composto por um conjunto de Funções. A função pela qual o programa começa a ser executado chama-se main.
  • Após cada cada comando em C deve-se colocar um ; (ponto-e-vírgula).
  • Um programa em C deve ser identado (um tipo de organização de código) para que possa ser lido com mais facilidade.

Como executar um programa em C

Enquanto você for praticando os conceitos mostrados aqui, você deve executar o seu código para ver se ele está funcionando corretamente!

Para fazer isso, você deve salvar o seu código com o nome que preferir e abrir o terminal.

Agora, caso não o tenha feito, instale o compilador, inserindo essa linha no terminal:

sudo apt install gcc

após isso, você deve entrar na pasta que você salvou o código dentro do terminal e inserir "gcc" e o nome do seu código com a extensão

(dica: use Tab para autopreencher o nome do documento que você está procurando).

gcc CodigoQueQueroCompilar.c

Nesse momento aparecerão os warnings e os erros do seu programa, se existirem.

Agora, para executar o código, só falta escrever:

./a.out

Variáveis e expressões matemáticas

Variáveis são nada mais nada menos que um tipo de armazenamento de dados que fica guardado na memória.

Em C, quando você declara uma variável, você deve especificar qual tipo de dado ela vai armazenar, como um inteiro (int), um número decimal (float) ou uma frase (string).

Para declarar a variável, basta escrever o seu tipo e o nome que deseja que ela tenha:

int main (){


int Var_NumInteiro;
float Var_NumDecimal;
string Var_Frase;

return 0;
}

Você pode criar várias variáveis de um memso tipo na mesma linha, ou atribuir um valor para ela na declaração:

int main (){


int Var_NumInteiro = 3;
float num1, num2, num3;
string Var_Frase = "Hello World!";

return 0;
}

As expressões matemáticas em c funcionam da mesma maneira que você já conhece, porém usando alguns símbolos específicos:

multiplicação: * (ex.: a*b)
divisão: / (ex.: a/b)
adição: +
subtração: -
resto da divisão: %
potência: ^ (ex.: a^2)

Estruturas condicionais

Chamamos de estruturas condicionais as instruções para testar se uma condição é verdadeira ou não.

A instrução IF é a estrutura de teste mais básica e é encontrada em todas as linguagens.
Ela permite executar uma série de instruções caso uma condição se realize. A sintaxe desta expressão é a seguinte:

if (condição){

*instruções que serao realizadas
se a condição for verdadeira*
}
*se a condição for falsa, as instruções dentro das chaves
não são executadas e o programa pula para essa área*

A condição deve ficar entre parênteses. É possível definir várias condições a serem cumpridas com os conectores AND e OR (&& e ||).

Exemplo:

if ((condição1) && (condição2)){

*instruções que serao realizadas
se as duas condições forem verdadeiras*
}
*se uma ou ambas das condições forem falsas, as instruções dentro das chaves
não são executadas e o programa pula para essa área*

if ((condição1) || (condição2)){

*instruções que serao realizadas
se uma das condições for verdadeira*
}
*somente se ambas das condições forem falsas, as instruções dentro das chaves
não são executadas e o programa pula para essa área*

Estruturas de Repetição

As estruturas de repetição consistem em blocos de código que devem ser executados um determinado número de vezes.

Às vezes é necessário que um programa que repita o mesmo código diversas vezes, e para isso usamos as instruções FOR e WHILE.

A instrução FOR é muito útil quando se sabe de antemão quantas vezes a repetição deverá ser executada.
Este laço utiliza uma variável para controlar a contagem do loop, bem como seu incremento.

for (valor_inicial; condição_final; valor_incremento){

*bloco de instruções que serão repetidas*
}

O valor inicial será o primeiro número da contagem do FOR; a condição final é o que determina quando o FOR vai parar
(como por exemplo, quando o valor de contagem chegar em 5); e o valor do incremento é o valor que será somado ao valor de contagem
a cada repetição. Por exemplo, se o FOR tiver os valores (i = 2; i < 7; i = i + 1), o valor inicial é 2, a cada repetição o valor
aumentará em 1 unidade, e o contador para quando ele chegar a 7, ou seja, o for repetirá 5 vezes.

A outra instrução de repetição, o WHILE, é usado quando é necessário que aquele bloco de texto se repita enquanto uma condição
é verdadeira. Por exemplo:

int i = 0;

while (i < 3){

*bloco de instruções que serão repetidas*
i = i + 1;
}

Nesse caso, o WHILE permanecerá em loop até que o valor de i deixe de ser menor que 3, ou seja, depois de 3 repetições,
já que o i está sendo incrementado com 1 unidade em cada repetição.
