    #include <stdio.h>
    void main()
        {printf("hello World!");}

    Este é o C, é uma das linguagens básicas da programação, sendo base para muitas outras linguagens.

    Ela é uma linguagem estruturada, é um paradigma que fundamenta que:

    1. As tarefas são executadas em sequência;
    2. Testes lógicos podem decidir a execução ou não de uma tarefa;
    3. Por meio destes testes, um trecho pode ser repetivél, mas não infinitamente.

    E assim, basicamente, o código C é executado.

  • Variáveis e expressões matemáticas

    Primeiro é bom saber as coisas que dão pra fazer no C, enquanto imprimir no terminal com printf pode ser divertido, as tarefas executadas em qualquer programa devem todas serem controladas de certa forma, e assim temos as variáveis.

    As variáveis são palavras decididas por você, que carregam alguma informação que será usada na execução da tarefa.

    #include <stdio.h>
    void main()
    {
        int numeroLegal, numeroDaora;
        int numeroLegal = 24;
        numeroDaora = numeroLegal + 1;
        printf(numeroDaora);
    }

    No exemplo duas variáveis de números inteiros são criadas, uma é definida como 24, enquanto a outra usa da informação guardada na primeira para ser definida como o inteiro 25.

    Muitas coisas são definidas por essas variáveis, neste exemplo nós sabiamos que número seria impresso, e podiámos ter só pedido para imprimir "25", mas na maioria das vezes, essa informação pode precisar mudar, às vezes diferentes partes do programa sejam executadas dependendo do que uma variável guarda ou não guarda, às vezes essa informação pode ser alterada por uma fonte externa, e essas mudanças só podem ser guardadas por meio destas váriaveis.

    Como você pode ter visto no exemplo anterior, outras ferramentas disponíveis para a manipulação de informação são as expressões matemáticas. O computador trabalha com 0s e 1s para funcionar, e para ele TODA informação é secretamente um número (sim, até uma linha de texto), então será bastante comum você encontrar no meio do seu código, e dentro de suas funções coisas como +, -, /, *, assim como outros elementos matemáticos mais complexos, como floor() que arredonda para baixo um número real, ou exp() que caucula o valor exponencial de um número. Outra expressão matemática comum de ser vista na programação é o =, ele empenha várias funções muito importantes dentro de um código, como no caso do exemplo, em que o = é responsável por guardar uma informação dentro de uma váriavel.

  • Estruturas Condicionais

    Estruturas condicionais são as estruturas responsáveis pela segunda parte do paradgma do C, elas definem que trechos do código são ou não são executados, garantindo mais controle sob a tarefa e permitindo interações mais especiais.

    A condição mais simples é chamada if(), essa estrutura executa as linhas de comando seguintes SE a condição inserida nela for verdadeira, é uma ação bem direta e simples, mas é extremamente importante para a criação de códigos em C. Por exemplo, se eu quiser saber se uma variável de número inteiro é positiva, eu posso escrever algo como:

    if(variávelBacanassa > 0)
        {printf("A vida é boa :)");}

    no corpo do meu código, assim ele imprimirá a mensagem positiva, apenas SE a variável for maior que 0. Mas e no caso de uma váriavel negativa? o que aconteceria se eu quisesse que a mensagem fosse diferente dependendo do estado da minha variável?

    if(variávelBacanassa > 0)
        {printf("A vida é boa :)");}
    else if(variávelBacanassa < 0)
        {printf(">:(((");}

    Com essas duas estruturas condicionais, nossa variável tem humor, e você pode começar a definir mais o que seu código pode fazer!

    Outras estruturas condicionais relevantes:

    • Switch - quando o código se divide em múltiplas condições diferentes da mesma váriavel;
    • Condicionais de curto-circuito ( && e || ) - também chamados and e or esses símbolos são usados entre condições dentro de uma mesma instância do if-then-else;
    • Ternário ( x ? y : z ) - forma mais curta de escrever um if-then-else, nele, o valor de y é retornado se a condição x for verdadeira, mas se não for, é retornado o valor de z.

  • Estruturas de Repetição

    Por último, temos uma estrutura igualmente importante para a linguagem C, muitas vezes é útil reutilizar partes do se código múltiplas vezes, mas escrever a mesma linha de texto toda hora é chato, cansativo e imprático. Essas estruturas servem para isso, repetir partes de um código, quantas vezes precisar.

    while(cansaco == false)
        {printf("oi :) /n");}

    É bom sempre lembrar que as condições para a repetição de código tem que ser capazes de parar a repetição eventualmente, um loop infinito é sempre perigoso e causará problemas na execução do seu código.
