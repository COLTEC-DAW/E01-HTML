Exemplo:

Variáveis e expressões matemáticas

No C, podemos criar variáveis escrevendo primeiro o tipo da variável (int, char, double) e setar um valor a ela com o "=", podendo ser desde um valor absoluto ou um resultado de uma expressão matemática.

Exemplo:

#include <stdio.h>
int main(){
  int a = 3 + 4;
return 0;
}

Estruturas condicionais

Estruturas condicionais são estruturas que dividem nosso código em diversos caminhos dependendo das condições. Entre as funções temos:
"if/else", que mudam o fluxo do código dependendo de uma condição;
"switch", onde o resultado da condição pode levar para vários caminhos;
e "break", que quebra a execução da função atual.

Exemplo:

#include <stdio.h>
int main(){
  int a = 4;
  if(a<5){
    printf("Funfou!");
  }
return 0;
}

Estruturas de repetição

As estruturas de repetição permitem que uma parte do código seja repetida até certo ponto, definido pelo coder. Entre essas funções temos:
"do/while", loop que a ação ocorre antes da checagem da condição;
"while", loop que ocorre depois da checagem da condição;
"for", loop que ocorre junto a um contador.

Exemplo:

#include <stdio.h>
int main(){
  while(1){
    printf("Yey!\n");   } return 0;
}

Como executar um programa em C

Para executar seu programa, você deve acessar o diretório onde o arquivo .c se encontra pelo Terminal (usando o comando "cd "NOME_DO_DIRETORIO""), digitar "gcc "NOME_DO_ARQUIVO.c"" e depois "./a.out".

cd Desktop
gcc teste.c
./a.out
