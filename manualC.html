Introdução

C é uma linguagem de programação compilada de propósito geral, estruturada, imperativa, procedural, padronizada pela ISO, criada em 1972, por Dennis Ritchie, no AT&T Bell Labs, para desenvolver o sistema operacional Unix (que foi originalmente escrito em Assembly). C é uma das linguagens de programação mais populares e existem poucas arquiteturas para as quais não existem compiladores para C. C tem influenciado muitas outras linguagens de programação, mais notavelmente C++, que originalmente começou como uma extensão para C.

#include <stdio.h>

int main(){
printf("Hello World\n");
return 0;
}

Variáveis e expressões matemáticas

Em C, as variáveis são declaradas antes de serem utilizadas, seguindo sempre o seguinte padrão:

//...
int a;
float inteiro;
char caractere;
//...

Com essas variáveis, é possível criar expressões matemáticas para que seu programa resolva, como por exemplo:

//...
int a = 5;
int b = 4;
int c = a+b;
printf("%i", c);
//...

Nesse exemplo, o resultado mostrado na tela será "9".

Estruturas condicionais

As estruturas condicionais servem para fazer seu programa tomar decisões. Por exemplo, se uma coisa acontecer, faça isso, caso contrário, faça aquilo.

Em C, temos as estruturas if, else if, else e switch case. A sintaxe das estruturas condicionais é bem simples:

if(condição){
//código
}
else if(condição){
//código
}
else{
//código
}


switch(variavel){
case 1: //código
break;
case 2: //código
break;
default: código
break;
}

Estruturas de repetição

As estruturas de repetição lembram um pouco as estruturas condicionais. Elas funcionam basicamente assim: enquanto uma condição for verdadeira, repita.
Na linguagem C, temos três estruturas de repetição: for, while e do while. A sintaxe dessas estruturas é:

for(estado inicial; condição para repetir; o que acontece a cada repetiçao){
//código
}
while(condição){
//código a se repetir
}
do{
//código a se repetir
}while(condição);

A diferença entre o while e o do while é que o primeiro vai checar se a condição é verdadeira antes de iterar pela primeira vez, enquanto que o segundo vai necessariamente iterar uma vez antes de checar.

Como executar um programa em C

Primeiramente, você deve compilar seu programa, utilizando uma linha de código no terminal:
gcc nomedoprograma.c
Isso vai gerar um arquivo executável, no caso com nome a.out. Para executar o programa, você deve digitar ./a.out na pasta onde se encontra seu programa. Pronto!
