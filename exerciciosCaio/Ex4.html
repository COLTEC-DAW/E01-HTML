Variáveis e expressões matemáticas

Uma variável é uma posição de memória que pode ser identificada através de um nome e pode ter seu conteúdo alterado por um comando de atribuição, após esta atribuição ela muda de valor.

Toda variavel em C tem um tipo e ele define os valores que a variavel pode armazenar. Observe na tabela abaixo:

Tipo Valores validos
char letras e símbolos: 'a', 'b', 'H', '^', '*','1','0'
int de -32767 até 32767 (apenas números inteiros)
float de -3.4 x 1038 até +3.4 x 10+38com até 6 dígitos de precisão
double de -1.7 x 10308 até +1.7 x 10+308com até 10 dígitos de precisão

É importante ressaltar que todas variaveis têm que ser inicializadas antes de serem usadas.

int x = 2 + 4; //variavel x foi criada e tem valor 6
int y = 3; //variavel y foi criada e tem valor 3
int z = x + y; //variavel z foi criada e tem valor de x(6) + y(3) = 9

Estruturas Condicionais

A condição na linguagem C é definida como uma expressão que pode ser verdadeira ou falsa. Se a condição for verdadeira os comandos que estão dentro do if serão executados, mas se a condição for falsa os comandos serão ignorados ou os comandos que estão dentro do else serão executados, se tiver o else do codigo

int x; //variavel x foi criada sem valor definido
int y = 3; //variavel y foi criada e tem valor 3
if(y = 3){ //se a variavel y tiver o valor igual a 3 os comandos dentro dessa chave serao cumpridos 
   x = 5; //como nesse caso a condicao e verdadeira x adquire o valor 5
} else { //se a variavel y nao tiver o valor igual a 3 os comandos dentro dessa chave serao cumpridos
x = 4; //se a condicao fosse falsa x iria adquirir o valor 4
}

Estruturas de Repetição

O comando while permite que um certo trecho de programa seja executado enquanto uma certa condição for verdadeira.

Primeiramente a condição é testada, se ela for falsa os comandos do bloco sao ignorados. Porem se for verdadeira os comandos sao executados e a condição é testada novamente. Veja o exemplo a seguir:

int x = 2; //variavel x foi criada e tem valor 2
int y; //variavel y foi criada sem valor definido
while(x=2){ //enquanto o valor da variavel x for igual a 2 os comandos dentro da chave serao cumpridos 
  y = 5; //enquanto a variavel x tiver o valor 2 a variavel y tera valor 5
}

Alem do comando while tambem existem os comandos for e do-while

Como executar um programa em C

Para compilar um programa no UBUNTU você deve primeiramente abrir o terminal. Depois você precisa compilar o programa, para isto escreva no terminal: gcc NomeDoCodigo.c

Se o seu progrma compilou, não teve erros e, de preferência, nenhum warning, execute-o digitando ./a,out NomeDoCodigo.c

gcc NomeDoCodigo.c
./a.out NomeDoCodigo.c
