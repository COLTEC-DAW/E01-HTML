Introdução

Criada pelo cientista da computação Dennis Ritchie, em 1972, a linguagem C foi derivada de outras duas: a BCPL e a Algol 68. Embora tenha sido pensada com o propósito exclusivo de ser usada no desenvolvimento de uma nova versão do sistema operacional Unix, hoje é aplicada nos mais variados tipos de projeto. A linguagem C ainda é uma das mais populares do mercado devido às diversas vantagens que apresenta.

Exemplo básico de código em C:

int main(){

printf("Hello World!\n");

return 0;

}

Variáveis e expressões matemáticas

Tipos principais de variáveis:

  • int = Número inteiro
  • float = Número racional
  • char = Caracter

Principais expressões matemáticas

  • + para soma
  • - para subtração
  • / para divisão
  • * para multiplicação
  • % para usar o resto de uma divisão
  • >, <, >=, <= para maior que, menor que, maior ou igual a, menor ou igual a

int main(){

int soma;

soma = 2 + 2;

return 0;

}

Estruturas condicionais

Usamos as estruturas condicionais para executar determinado código caso o parâmetro seja verdadeiro

int main(){

int numero;

printf("Digite um numero: ")

scanf("%i", &numero);

if(numero == 0){

printf("O numero digitado eh 0");

}

else if(numero == 2){

printf("O numero digitado eh 2");

}

else{

printf("O numero digitado nao eh 0 nem 2");

}

return 0;

}

O código só imprime "O numero digitado eh 0" SE a variável numero tiver valor 0.

Usamos o else if quando queremos adicionar outra condição a ser verificada.

O código só entra no else caso nenhuma das condicionais anteriores for verdadeira.

Estruturas de repetição

WHILE - O código é executado enquando a condicional for verdadeira.

int main(){

int i = 0, limite = 10;

while(i != limite){

printf("Valor de i = %d", i);

i = i + 1;

}

return 0;

}

FOR - Estrutura (variavel; limite; iterização)

int main(){

int limite = 10

for(i = 0; i <= limite; i++){

printf("Valor de i = %d", i);

}

}

Como exeuctar um programa em C

  • Digitar o código no seu editor de preferência: VS, Code blocks, Sublime...
  • Compilar e rodar
  • Lembre-se de encerrar a compilação antes de rodar novamente!
