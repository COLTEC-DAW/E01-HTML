Manual de Programação

Introdução

  • C é uma linguagem de programação compilada de propósito geral, estruturada, imperativa, procedural, padronizada pela Organização Internacional para Padronização, criada em 1972 por Dennis Ritchie na empresa AT&T Bell Labs para desenvolvimento do sistema operacional Unix.

  • C é uma das linguagens de programação mais populares e existem poucas arquiteturas para as quais não existem compiladores para C. C tem influenciado muitas outras linguagens de programação (por exemplo, a linguagem Java), mais notavelmente C++, que originalmente começou como uma extensão para C.

Variáveis e expressões matemáticas

  • Em C temos as variaveis principais:
    1. Int para numeros inteiros
    2. Char para caracteres
    3. Float para numeros reais

  • Em C temos as expressoes principais:
    1. + para soma
    2. - para subtração
    3. / para divisão
    4. % para resto da divisão
    5. = para atribuição

    Estruturas Condicionais

  • Em c usamos if(), else if() e switch case() como estruturas condicionais.
    1. if() é o mais basico para uma condiçao.
      #include < stdio.h>
      #include < stdlib.h>
      if(1>2){
      printf("1 é maior que 2")
      }

  • O else if() é uma segunda condição para o if()
    1. #include < stdio.h>
      #include < stdlib.h>
      if(1>2){
      printf("1 é maior que 2")
      }else if(3>2){
      printf("3 é maior que 2")
      }

  • O switch case() é usado para usar varias condiçoes como no caso de um menu
    1. #include < stdio.h>
      #include < stdlib.h>
      int menu
      printf("Digite um numero de 1 a 5")
      scanf("%d", &menu)
      switch case(menu){
      case 1:
      printf("Voce digitou 1")
      break;
      case 2:
      printf("Voce digitou 2")
      break;
      case 3:
      printf("Voce digitou 3")
      break;
      case 4:
      printf("Voce digitou 4")
      break;
      case 5:
      printf("Voce digitou 5")
      break;
      default:
      printf("Invalido")
      break;
      }

    Estruturas de Repetição

  • Temos 2 estruturas de Repetição for() e while()
    1. For()
      #include < stdio.h>
      #include < stdlib.h>
      for(int x = 0, x < 100, x++){
      printf("%d", x)
      }

    1. while()
      #include < stdio.h>
      #include < stdlib.h>
      int x;
      while(x<100){
      printf("%d", x)
      }
