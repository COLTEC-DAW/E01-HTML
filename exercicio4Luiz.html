Tutorial completo C!

Introdução

Um programa em C é composto por um conjunto de Funções. A função pela qual o programa começa a ser executado chama-se main. Após cada cada comando em C deve-se colocar um ; (ponto-e-vírgula). Um programa em C deve ser Identado para que possa ser lido com mais facilidade.


Variáveis e expressões matemáticas

Variável é um local reservado na memória para armazenar um tipo de dado. Toda variável deve ter um identificador, ou seja um nome. Além de ter um nome, a variável também precisa ter um tipo. O tipo de dado de uma variável determina o que ela é capaz de armazenar. Em C, fazer operações matemáticas simples é bastante fácil e intuitivo. Por exemplo, se quisermos que uma variável contenha o resultado da conta 123 + 912, fazemos assim: var = 123 + 912; Os operadores aritméticos básicos são 5: + (adição), - (subtração), * (multiplicação), / (divisão) e % (resto de divisão inteira).

celsius = (fahrenheit - 32) * 5.0 / 9.0;
forca = massa * aceleracao;
i = i + 1;


Estruturas condicionais

Quando um programa executa ou deixa de executar instruções com base no valor de uma condição, o programa realiza um processamento condicional. Para executar um processamento condicional, um programa precisa utilizar o comando if. Todo comando if requer uma condição. O valor de uma condição pode ser verdadeiro ou falso. Em C, não existe um tipo de dados específico para representar valores lógicos (V ou F).

if (condição realizada) {
lista de instruções;
}


Estruturas de repetição

Esta estrutura de repetição, garante que o bloco de instruções seja executado no mínimo uma vez, já que a condição que controla o laço é testada apenas no final do comando. A diferença entre o comando while e o do…while é justamente o local onde a condição que controla o laço é testada. No comando while a condição é testada antes do bloco de instruções, e caso a condição seja falsa a repetição não será executada. No do…while o bloco de comandos é executado pelo menos uma vez de forma obrigatória, independente do resultado da expressão lógica.

while (condição realizada) {
lista de instruções;
}


Como executar um programa em C

Na janela do prompt de comando do desenvolvedor, digite cd c:\ para alterar o diretório de trabalho atual para a raiz da unidade C:. Em seguida, digite md c:\simple para criar um diretório e, em seguida, digite cd c:\simple para alterar para esse diretório.
