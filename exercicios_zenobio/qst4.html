Introdução

Este é um manual que explica as instruções básicas da linguagem C.

Variáveis e expressões matemáticas

Uma variável é uma possição de memória que pode ser identificada através de um nome. Elas podem ter seu conteúdo alterado por um comando de atribuição, exemplificado abaixo:

int a = 3;

Todas as variáveis em C tem um tipo que define que tipo de valor ela pode armazenar, cada um deles ocupando uma certa quantidade de memória do sistema. São eles:

Tipo   Valores Válidos
char letras e símbolos:   'a',  'b', 'H', '^', '*','1','0'
int de -32767 até 32767 (apenas números inteiros)
float de -3.4 x 1038 até +3.4 x 10+38com até 6 dígitos de precisão
double de -1.7 x 10308 até +1.7 x 10+308com até 10 dígitos de precisão

Fonte: http://www.inf.pucrs.br/~pinho/LaproI/IntroC/IntroC.htm

As expressões matemáticas básicas são realizas pelo símbolos +, -, *, / e podem ser realizadas tanto entre variáveis como valores e armazenadas em variáveis.

int a = 4;
int b = 2;
int c = a + b; //c recebe o valor 6
int c = a * b; //c recebe o valor 8
int c = a - b; //c recebe o valor 2
int c = a / b; //c recebe o valor 2

Estruturas Condicionais

Estruturas condicionais são instruções da linguagem C capazes de decidir se parte do código será ou não executado.

int a = 3;
if(a == 3){} //executa o código contido nas chaves caso seja verdadeiro
else{} //executa outro códgo contido nas chaves caso a condição anterior seja falsa

No código acima, a instrução if define que o código que ela guarda entre as chaves somente deve ser executado caso a variável 3. Entre parênteses está estabelecida a condição citada.

Para comparar valores ou variáveis, são utilizados os comparadores == (igual a), != (diferente de), <(menor que), > (maior que).

A Instrução switch efetua vários testes de valores sobre o conteúdo de uma mesma variável. Esta conexão condicional simplifica muito o teste de vários valores de uma variável, pois a operação teria sido complicada (mas possível) com varios ifs.

int a = 3;
switch (a) {
  case 1: //se a for igual a 1, executa um codigo especifico
  break;

  case 2: //se a for igual a 2, executa outro codigo especifico
  break;

  case 3: //se a for igual a 3, executa outro codigo especifico
  break;

  default: //se não for igual nenhum dos testes anteriores, executa outro codigo especifico
}

A condição, no caso do switch, é definida pela igualdade da variável entre parênteses com os valores dos casos definidos. O código executado pela condição que for verdadeira é definido após os dois pontos.

Estruturas de Repetição

Estruturas de repetição são instruções da linguagem C capazes de decidir não somente se parte do código será ou não executado, como, também, de executar novamente o mesmo código diversas vezes até que a condição definida seja falsa.

do{}while(a == 3); //verifica a condição após executar uma vez o código while(a == 3){}; //verifica a condição antes de executar o código pela primeira vez

A função while define um código que deve ser executado repetidamente até que a condição definida seja falsa

for(a = 3; a < 6; a = a + 1){}

Já função for define um código que deve ser executado repetidamente inicializado em meio a um contexto, em que, também, é definida uma váriavel e uma instrução comum executada a cada repetição até que esta seja falsa.

Como executar um programa em C

Para executar um programa aem C, salve o arquivo na sua pasta pessoal e abra o terminal. Utilize os comandos abaixo na ordem descrita:

  1. gcc nomeDoArquivo.c -o nomeDoArquivo.bin
  2. ./nomeDoArquivo
