Manual Básico da Linguagem C

Introdução

C é uma linguagem de programação compilada de propósito geral, estruturada, imperativa, procedural, padronizada pela ISO. Ela foi criada em 1972, por Dennis Ritchie, para desenvolver o sistema operacional Unix (que foi originalmente escrito em Assembly). C é uma das linguagens de programação mais populares e existem poucas arquiteturas para as quais não existem compiladores para C. Ela tem influenciado muitas outras linguagens de programação, mais notavelmente C++, que originalmente começou como uma extensão para C.

Primeiro incluimos a biblioteca que vamos utilizar e depois colocamos a main e depois declaramos as variáveis e é ali dentro que vamos desemvolver o programa. Aqui temos a estrutura básica de um código C:

Obs.: não deve haver espaço entre os sinais de maior e/ou menor e o nome da biblioteca que devemos incluir.

//Isto é um comentário
#include < stdio.h >
main(){
}

Variáveis e expressões matemáticas

Temos basicamente três tipos principais de variáveis em C.

Tipo Explicação Exemplo prático
int Armazena valores inteiros. 1, 500, -70
float Armazena números decimáis, positivos e negativos. 0.3659, -25.156
char Armazena letras, nomes, caractéres em geral. m, v, #, *

Exemplo:


            #include < stdio.h >

            main(){

                int num;

                float troco;

                char nome;

            }

Estruturas Condicionais

Temos na linguagem C três tipos de estruturas que são caracterizadas como sendo de condição:

if -> é a estrutura de teste mais básica , ela permite executar uma série de instruções caso uma condição se realize ou não. Exemplo:


            #include < stdio.h >

            main(){

                if (condição realizada) {

                    lista de instruções;

                }

            }

if...else -> permite executar outra série de instruções em caso do não cumprimento da condição. A sintaxe desta expressão é a seguinte:


            #include < stdio.h >
    
            main(){
    
                if (condição realizada) {

                    lista de instruções

                }

                else {

                    outra série de instruções

                }

            }

switch -> efetua vários testes de valores sobre o conteúdo de uma mesma variável. Esta conexão condicional simplifica muito o teste de vários valores de uma variável, pois esta operação teria sido complicada (mas possível) com ifs imbricados. Sua sintaxe é a seguinte: 


            #include < stdio.h >

            main(){

                switch( opção ){

                    case 1:

                        lista de comandos

                    break;

                    case 2:

                        lista de comandos

                    break;
                    case 3:
                        lista de comandos

                    break;

                    default:
                    lista de comandos caso nenhuma das opções anteriores tenha sido escolhida

            }

Estruturas de Repetição

Temos duas estreuturas de repetição em C. São elas:

for -> executa várias vezes a mesma série de instruções, ou seja, é um loop. Exemplo de sua sintaxe: 


            #include < stdio.h >

            main(){

                for (contador; condição; modificação do contador) {

                        lista de instruções;

                }

            }

Contador é responsável por dizer por onde o loop deve ser avaliado, a condição é quem vai dizer qual a condição ser avaliada e modificação do contador vai dizer se o loop vai ir somando ou subtraindo a partir da variável dada pelo contador.

while -> representa outro meio de executar várias vezes a mesma série de instruções. Esta instrução executa a lista de instruções enquanto a condição se realiza. A sintaxe desta expressão é a seguinte: 


            #include < stdio.h >

            main(){

                while (condição realizada) {

                    lista de instruções;

                }

            }

Como executar um programa e C

Para executar seu programa em C, você deve ir até o terminal e digitar os seguintes comandos:

gcc nome do seu arquivo .c
.\a.out

Pronto, seu programa estará rodando!!
