Introdução:

Considerada uma linguagem de alto nível genérica, a C pode ser usada em diversos tipos de projeto, como a criação de aplicativos, sistemas operacionais, drivers, entre outros. Trata-se de uma linguagem estruturada que se tornou muito popular nos anos 80 — tanto que é difícil encontrar arquiteturas para as quais não existam compiladores para a C, o que garante o seu elevado nível de portabilidade.

Uma das grandes vantagens dessa linguagem é a capacidade de gerar códigos rápidos, ou seja, com um tempo de execução baixo. Além disso, a programação em C é bastante simplificada, pois sua estrutura é simples e flexível. Tendo isso em mente, podemos dizer que as principais características da linguagem C são:

  • portabilidade;
  • geração de código eficiente;
  • simplicidade;
  • confiabilidade;
  • facilidade de uso;
  • regularidade.

Exemplo de um código em C que diz "Hello world".

    #include < stdio.h>
    #include < stdlib.h>
    #define sucesso 0

    int main(){

        printf(" Hello world ");
        
        return sucesso;
    }

Variáveis e expressões matemáticas:

Na programação utilizamos dois conceitos muito importantes, que vão te ajudar a fazer seus próprios códigos em breve:

  • Variaveis: São "lugares" onde armazenamos dados. Essas variaveis indicam um lugar na memória onde estão as informações que precisamos.
    • Sintaxe: Armazena: Código:
    int números inteiros int NumeroI = 5; //A variavel "NumeroI" recebe o valor 5
    float números decimais float NumeroD = 5.2; //A variavel "NumeroD" recebe o valor 5.2
    double números decimais double NumeroDE = 5; //A variavel "NumeroDE" recebe o valor 5
    (possui uma precisão maior que o float)
    char caracteres char Letra = "E"; //A variavel "Letra" recebe o caracter "E"

  • Operações: São como as vistas nas aulas de matemática, segue alguns exemplos:
    • Caracter: Operação: Código:
    + soma int Soma = 5 + 8; //Atribui a variavel "Soma" o valor 13(5+8)
    - subtração int Sub = 5 - 8; //Atribui a variavel "Sub" o valor -3(5-8)
    / divisão int Div = 16 / 8; //Atribui a variavel "Div" o valor 2(16/2)
    * multiplicação int Mult = 5 * 8; //Atribui a variavel "Mult" o valor 40(5*8)
    % resto int Res = 10 % 8; //Atribui a variavel "Res" o valor 2(10%8 Resto da divisão 10/8) 
    #include < stdio.h>
    #include < stdlib.h>
    #define sucesso 0

    int main(){

        int Numero01 = 5;
        int Numero02 = Numero01 * 7;
        float Decimal01,Decimal02;

        Decimal01 = Numero01 % Numero02;
        Decimal02 = Decimal01 / Numero01;

        return sucesso;
    }

Estruturas Condicionais:

Para realizar determinadas tarefas vamos precisar verificar se algum dado esta correto, até mesmo para validar uma senha! Em C possuimos uma estrutura que faz essa verificação e algumas variações dela para outras situações.

Sintaxe: Função:
if Executa um bloco de informações se uma informação for verdadeira
else Executa um bloco de informações caso a estrutura 'if' anterior não seja verdadeira
else if Seguinte a uma estrutura 'if' que irá fazer outra checagem da informação
switch Simplifica o código, substituindo uma sequência de 'else if's


Estruturas de Repetição:

As vezes precisamos executar repetidas vezes um mesmo trecho do código, e nem sempre vamos saber o número exato de vezes que teremos que repetir =/...
Mas para resolver esse problemas temos algumas estruturas que nos auxiliam em cada situação.

Estrutura: Função:
for Repete um bloco de código por N vezes
while Repete um bloco de código enquanto uma condição for verdadeira
do while: Similar ao 'while' porem checa a condição ao fim do bloco de código

Como executar um programa em C:

Há varias maneiras de se executar um código, pode-se usar uma IDE, compiladores onlinde e/ou utilizar o proprio terminal(como será explicado a seguir).

Configurando o compilador na sua máquina:

  • Caso use o linux pule as duas próximas etapas!
  • Inicialmente baixe e instale o mingw;
  • Após a instalação, configure o comando gcc como uma variavel de seu sistema;

Executando seu programa(Hello.c):

  • Abra o prompt de comando e navegue até o diretório onde esta seu código;
  • Digite no terminal o seguinte comando: "gcc Hello.c"
  • Seu código ja está compilado, agora basta roda-lo para ver sua execução;
  • Digite no terminal o seguinte comando: "./a.exe" e seu código irá ser executado;

Exemplo de um código e sua saída: 

Nome do arquivo: Hello.c
#include < stdio.h>
#include < stdlib.h>
#define sucesso 0

int main(){

    printf("Hello world!\n");

    return sucesso;
}
Execução do arquivo:
cmd: gcc Hello.c
cmd(Saída): Hello world!
