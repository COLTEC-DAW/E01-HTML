Variáveis e expressões matemáticas

As variáveis são locais onde são armazenados alguns componentes como valores, letras(char),etc. Cada variável possui um tipo que determina que tipo de componente pode ser armazenado nela, alguns exemplos são:

  • Int: a variável só pode guardar números de valor inteiro
  • Float: armazenamentos de números com vírgula
  • Char: guarda uma letra

As variáveis são usadas para armazenar os dados necessários para o funcionamento do programa. Com elas são feitas algumas operações, por exemplo, operações algébricas. Dentre elas é possível somar, subtrair, multiplicar, dividir, exponenciar, radiciar, etc.

Exemplos de expressões matemáticas

Nesse exemplo estarei usando números inteiros, no entanto é perfeitamente possível usar float. Para exemplicar estarei usando duas variáveis que guardam números inteiros, variável "a" e "b".

Soma:

int a, b; ---O programa começa com a declaração das varáveis que serão usadas!
int soma; ---É necessário, nesse caso, uma variável que guarde o resultado da soma.

soma = a + b; --- Então, fazemos a atribuição da variável soma como sendo o resultado da soma de "a" e "b".

Esse é um exemplo clássico de operações algébricas

Subtração:

int a, b;
int sub;
sub = a - b;

Grande parte das operações algébricas seguem essa lógica, porém há algumas que se diferenciam!. Há algumas operações que só podem serem realizadas com funções específicas, como é o caso da exponenciação e radiciação.

Exponenciação:

int a, b;
int exp;
exp = pow(a, b); ---A função responsável pela exponenciação é pow, sendo o primeiro número no parênteses a base e o segundo o grau.

Radiciação:

int a, b;
int rad;
exp = sqrt(a, b); ---A função sqrt realiza a radiciação, sendo o primeiro número no parênteses a raiz e o segundo o expoente.

Estruturas Condicionais

As estruturas condicioinais são usadas quando há uma necessecidade de limitar uma ação no programa ou o própio prosseguimento do programa à uma condição
Exemplo: Se meu saldo é maior que a quantidade que eu quero sacar na minha conta no banco é possível realizar a operação. Senão a operação é barrada.
Alguns exemplos de operações condicionais são: if, else-if. Vou exemplicar e explicar a ultilização de cada um.

If:

Ele é utlizado quando há uma necessidade de executar uma determinada declaração de acordo com uma condição específica.
Exemplo:

int a, b;
int x;
if(a>b){
  x=4;
}

Repare que a declaração só será verdadeira se a for maior que b, caso contrário o programa continua seguindo e ignora o if.

Else:

O else é usado quando é necessária a análise de mais de uma condição condição.
Descrião:

if(declaração 1){};
else {declaração 2};

Repare que caso o primeiro if seja falso o programa segue diretamente para o else e o executa, a explicação básica para o else seria: Faça isso(if), senão faça essa outra coisa(else).

If-Else Aninhados

São vários if-else atrelados que são ultilizados quando há necessidade de especificar vários casos condicionais.
Descrição:

if(declaração 1){}
  else{declaração 2}
if(declaração 1){}
  else{declaração 2}

Estruturas de repetição

Estruturas de repetição são loops utilizados para realizar uma determinada ação por uma quantidade de vezes determinadas ou até que uma condição se torne falsa. Alguns exemplos dessas estruturas são o while, do-while e o for.

While:

Realiza uma ação enquanto uma determinada ação é verdadeira ou enquanto falsa.

Exemplo:


int i=0; while(i<5){ --- Enquanto "i" for maior que 5 o bloco while vai executar o comando que está dentro dele.
  x=x*y;
  i++;--- Repare que inicialmente i é iniciado em 0 e ao longo de cada loop ele é incrementando em uma unidade(++). Logo, o bloco será executado 5 vezes.
}

É necessário que o loop tenha uma limitação bem definida, caso o contrário o bloco pode ficar sendo executado até consumir toda a memória Ram do computador.


Do-While:

Possui o mesmo princípio que o while, a diferença é que o do-while vai executar a ação pelo menos uma vez a mais que o while sendo que nesta repetição a condição não será levada em conta.


Exemplo


do{
  Comando }while(condição)

For:

É um loop com delimitação definida assim como os outros, o diferencial sua sintaxe. Na sintaxe há a regra de controle que contém um contador que é inicializado em algum número e vai sendo decrementado ou incrementado.
Exemplo:

for(i=0;i<5;i++){   Comando }

Execução

Para executar o programa há algumas etapas, no linux o compilador será o terminal. No terminal digite gcc, que é o programa responsável pela compilação, seguido do nome do arquivo do programa na sua pasta pessoal.

$ gcc meu_arquivo.c

Isso vai gerar um arquivo executável com a extensão .out, o qual será responsável pela execução do seu programa.

$ ./meu_arquivo.out
