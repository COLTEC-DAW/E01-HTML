Manual de Programação

Introdução

C é uma linguagem de programação compilada de propósito geral, estruturada, imperativa, procedural, criada em 1972 por Dennis Ritchie na empresa AT&T Bell Labs para desenvolvimento do sistema operacional Unix (originalmente escrito em Assembly). C é uma das linguagens de programação mais populares e existem poucas arquiteturas para as quais não existem compiladores para C.

Variáveis e expressões matemáticas

Variáveis

Variável é um local reservado na memória para armazenar um tipo de dado. Toda variável deve ter um identificador, ou seja um nome. Além de ter um nome, a variável também precisa ter um tipo. O tipo de dado de uma variável determina o que ela é capaz de armazenar.

Tipos de Variáveis

Tipos Armazena
Int Valores Inteiros
Double/Float Pontos flutuantes, ou seja, valores com casas decimais
Char Caracter

Existem outros diversos tipos que podem ser utilizados em C!

Estruturas Condicionais

If-Else

Uma instrução if-else controla a ramificação condicional. As instruções do corpo da estrutura serão executadas somente se a condição for avaliado como um valor diferente de zero (ou true ). Caso contrário, a instrução a seguir será ignorada e, se houver else, a instrução após a else execução será executada. Exemplo: 

                    if(condição){
                        bloco de instruções
                     }
                    else{
                        bloco de instruções
                     }

Switch

Uma instrução switch efetua diversos testes de valores com o conteúdo de uma mesma variável. Assim, simplifica o teste de vários valores de uma variável. O switch é muito utulizado em menus. Exemplo: 

                    switch(variável) { 
                        case "valor1":
                            bloco de instruções
                        break;
                        case "valor2":
                            bloco de instruções
                        break;
                        case "valor3":
                            bloco de instruções
                        break;
                        default:
                            bloco de instruções
                    }

Estruturas de Repetição

For

Uma instrução for repete várias vezes determinado bloco de instruções, formando assim um loop, seguindo determinada condição para sua repetição. Exemplo: 

                    for (declaração do contador; condição; incremento do contador) {
                        bloco de instruções
                    }

While

Uma instrução while também forma um loop, enquanto alguma condição ainda é válida. Exemplo: 

                    while (condição){
                        bloco de instruções
                    }

Como executar um programa em C

O gcc - compilador utilizado na linguagem C, é uma ferramenta que já vem instalada na maioria dos sistemas. Caso o programa desejado seja executado apenas com um arquivo, você pode utilizar o comando abaixo, no terminal:

$ gcc (nome_do_arquivo).c -o (nome_do_executavel)

É comumente utilizado o nome do arquivo executavel o mesmo do nome original do arquivo, por facilidade, mas fica ao seu critério. No caso do Windows, é necessário a extensão.exe para reconhecer um arquivo executável.

Agora com o arquivo executável, por fim, para executar o programa, a maneira mais conhecida é utilizar o comando abaixo no terminal:

$ ./prog
