Introdução

C é uma linguagem de baixo nivel que existe há mais de 50 anos. Ela é utilizada para escrever de tudo, desde programas simples até sistemas operacionais completos.

Variaveis e expressões matemáticas

C suporta as seguintes variaveis primitivas básicas:

int numero inteiro
float numero com parte fracionaria
double float com o dobro de prescisão
char um unico caractere

E os seguintes operadores:

- e + subtração e adição
* e / multiplicação e divisão
% modulo da divisão (resto)
-- decremento (valor - 1)
++ incremento (valor + 1)

As operações realizadas em C seguem as ordens de precedência padrões da matemática (*, / e % realizados antes de + e -)
Também é possivel utilizar de parêntesis para organizar a expressão da forma que desejar

    Exemplos:

  • int i = 5;
    int j = 7;
    int y = j + i;
    Valor final de y = 12

  • float i = 5;
    float j = 2;
    int y = i % j;
    y--;
    Valor final de y = 0

  • float i = 10;
    float j = 5;
    int y = (i + j) * 3;
    Valor final de y = 45

Estruturas condicionais

Condicionais são estruturas usadas para realizar ações dependendo se determinada condição for verdadeira ou falsa

  • if (expressão) {
     (realiza algo se a expressão for verdadeira)
    }else{
     (realiza algo se a expressão não foi verdadeira)
    }

Para determinar tais expressões são usados alguns operadores expecificos

== se os valores de ambas as variaveis forem iguais, a expressão é verdadeira
!= se os valores de ambas as variaveis forem diferentes, a expressão é verdadeira
> e >= se o valor da variavel a esquerda for maior/ maior ou igual, a expressão é verdadeira
< e <= se o valor da variavel a esquerda for menor/ menor ou igual, a expressão é verdadeira

    Exemplos:

  • if (i >= 60){
     printf("eh maior");
    }

  • if (i != j){
     printf("eh diferente");
    }else{
     printf("eh igual");
    }

Estruturas de repetição

Estruturas de repetição são realizadas repetidamente enquanto a expressão for verdadeira

  • while (expressão) {
     (realiza algo enquanto a expressão for verdadeira)
    }
  • for (valorInicial; condição; incrementoDoValor) {
     (realiza algo enquanto a expressão for verdadeira)
    }

    Exemplos:

  • while (i < 5){
     printf("%d", i);
     i++;
    }

  • for (i = 0; i < 7; i++){
     printf("%d",i);
    }

Como executar um programa em C

Existem diversas formas de executar um programa em C, mas a forma mais facil é definitamente baixando uma IDE ou utilizando uma na internet

No Visual Studio, por exemplo, na caixa de ferramentas no topo da tela existe o botão Play e, caso o seu programa não apresente nenhum erro, seu programa ira ser executado em um terminal que sera aberto para seu programa
