Manual de C

Introdução

C é uma linguagem de programação compilada de propósito geral, estruturada, imperativa, procedural, padronizada pela ISO, criada em 1972, por Dennis Ritchie, no AT&T Bell Labs, para desenvolver o sistema operacional Unix (que foi originalmente escrito em Assembly). C é uma das linguagens de programação mais populares e existem poucas arquiteturas para as quais não existem compiladores para C. C tem influenciado muitas outras linguagens de programação, mais notavelmente C++, que originalmente começou como uma extensão para C.

Variáveis e Expressões Matemáticas

Dentro da linguagem C, nos temos alguns tipo de variáveis que podem ser criadas. Elas podem ser dos seguintes tipos:

  • int (Números inteiros)
  • float (Números com decimais)
  • double (Decimais com uma precisão maior)
  • char (Caracteres)
Essas variáveis são criadas no seguinte estilo:
    "tipo" "nome_da_variável";

Somente são aceitos nomes iniciados por letras ou _ (underline).Exemplos:

Válidos: Inválidos:
int variavel = 5; int 4numero = 0;
char _var = ' '; char 3 = 'a';
int __idade__ = 19; int 2_digitos = 45;

Além disso, é possivel realizar expressões matemáticas no C também. Os operadores matemáticos existentes são: "+", "-", "/" e "*"
Todos esses simbolos são bem conhecidos, principalmente por quem usa calculadora no computador, então não é necessária a explanação deles.
É possivel atribuir valores à variáveis através desses operadores assim como nos exemplos abaixo:

  • int numero = 4 + 5;
  • float decimal = 2.0/3.5;

Estruras Condicionais

Chamamos de estrutura condicional as instruções para testar se uma condição é verdadeira ou não. Para entendê-las melhor, precisamos do conceito de bloco.
Quando você quiser agrupar várias instruções, você pode criar o que chamamos de bloco, isto é, um conjunto de instruções (seguidas, respectivamente, por pontos e vírgulas) entre chaves {e}.

Instrução "if"

A instrução "if" é a estrutura de teste mais básica, ela é encontrada em todas as linguagens (com sintaxes diferentes...). Ela executa uma série de instruções caso uma condição se realize.

Sintaxe:
  if (condição realizada) {
    lista de instruções;
  }

Instrução if ... else

A instrução if em sua forma básica só testa uma condição ou, na maioria das vezes, gostaríamos de poder escolher as instruções a serem executadas em caso do não cumprimento da condição...
A expressão if ... else executa outra série de instruções, em caso do não cumprimento da condição.

Sintaxe:
  if (condição realizada) {
    lista de instruções;
  }
  else {
    outra série de instruções;
  }

Instrução switch

A Instrução switch efetua vários testes de valores sobre o conteúdo de uma mesma variável. Esta conexão condicional simplifica muito o teste de vários valores de uma variável, pois esta operação teria sido complicada (mas possível) com ifs aninhados.

Sintaxe:
  switch(variável){
    case 1:
      instruções executadas para a variável = 1;
    break;
    casa 2:
      instruções executadas para a variável = 2;
    break;
    casa 3:
      instruções executadas para a variável = 3;
    break;
    default:
      instruções executadas para qualquer outro valor de variável;
  }

Estruras de Repetição

Comumente chamadas de loops, são estruturas que executam várias vezes a mesma série de instruções até que uma condição não seja mais realizada... Às vezes, chamamos estas estruturas de instruções repetitivas ou iterações.
A forma mais comum de fazer um loop é criar um contador (uma variável que se incrementa, ou seja, que aumenta de 1 a cada volta do loop) e parar o loop quando o contador excede um determinado valor.

Instrução "for"

A instrução while representa outro meio de executar várias vezes a mesma série de instruções. A sintaxe porém é diferente do for, não possuindo a especificar da variável contadora e a incrementação dentro dos parenteses. A única coisa dentro dos parenteses é a condição.

Sintaxe:
  for (contador; condição; modificação do contador) {
    lista de instruções;
  }

Instrução while

Executa várias vezes a mesma série de instruções: é um loop! e em sua sintaxe, basta especificar o nome da variável que serve de contador, a condição sobre a variável pela qual o loop para (basicamente um limite) e, finalmente, uma instrução que incrementa (ou decrementa) o contador.

Sintaxe:
  while (condição realizada) {
   lista de instruções;
  }

Executando o programa

Após criar o programa, salve-o com a extensão .c. Exemplo: "nomedoprograma".c
Abra o terminal, entre na pasta em que o programa foi salvo e digite o comando a seguir:
gcc "nomedoprograma".c -o "nomedoprograma".bin

Para executar basta digitar: ./"nomedoprograma".c
