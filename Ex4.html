Aprendendo a Linguagem C

Introdução:

C é uma linguagem de programação compilada de propósito geral, estruturada, imperativa, procedural, padronizada pela ISO, criada em 1972, por Dennis Ritchie para desenvolver o sistema operacional Unix (que foi originalmente escrito em Assembly).

C é uma das linguagens de programação mais populares e existem poucas arquiteturas para as quais não existem compiladores para C. C tem influenciado muitas outras linguagens de programação, mais notavelmente C++, que originalmente começou como uma extensão para C.

Para mais informações clique aqui.

Variáveis e expressões matemáticas:

A linguagem C possui tipos determinados de variáveis. As principais podem ser inteiras (int), reais (float), caracteres (char).

#include <stdio.h>
int main(){

int nome_da_variavel1;
float nome_da_variavel2;
char nome_da_variavel3;
}

Estruturas condicionais:

Chamamos de estrutura condicional as instruções para testar se uma condição é verdadeira ou não.

IF

Ela executa uma série de instruções caso uma condição se realize.

if(condição realizda){
lista de instruções
}

IF...ELSE

só testa uma condição ou, na maioria das vezes, gostaríamos de poder escolher as instruções a serem executadas em caso do não cumprimento da condição... A expressão if ... else executa outra série de instruções, em caso do não cumprimento da condição.

if (condição realizada) {
lista de instruções
}
else {
outra série de instruções
}

Switch

A Instrução switch efetua vários testes de valores sobre o conteúdo de uma mesma variável.

switch (variável) {
casa Valor1 :
Lista d instruções;
break;
casa Valor2 :
Lista d instruções;
break;
default:
Lista d instruções;
}

Estruturas de repetição:

Os loops são estruturas que executam várias vezes a mesma série de instruções até que uma condição não seja mais realizada.

FOR

for (contador; condição; modificação do contador) {
lista de instruções;
}

WHILE

while (condição realizada) {
lista de instruções;
}

Como executar um programa em C:

Com o gcc já instalado no seu sistema, é muito simples usá-lo para compilar programas em C. Se o programa consistir de um único arquivo, você pode simplesmente executar este comando no terminal:

$ gcc prog.c

Para executar basta realizar o seguinte comando:

./a.out
