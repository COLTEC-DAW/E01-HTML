    • Variáveis são, basicamente, uma posição na memória da máquina que identificamos por um nome (identificador). Na linguagem C, todas as variáveis precisam ser declaradas no início do programa, antes de serem usadas.

    Elas também devem receber um tipo, que vai determinar quais espécies de valores cada variável pode armazenar. Por fim, uma variável pode ter seu conteúdo alterado a qualquer momento.

    Em C, os tipos de variáveis são:

  • char: armazena letras e símbolos;
  • int: armazena números inteiros;
  • float: armazena números reais;
  • double: representam números de ponto flutuante de precisão dupla, onde precisam duas vezes a quantidade de memória das variáveis float fornecendo 15 dígitos, sendo o dobro da precisão de variáveis float.

    • Estruturas condicionais

    Usamos as estruturas condicionais para executar determinado código caso o parâmetro seja verdadeiro

    int main(){

    int numero;

    printf("Digite um numero: ")

    scanf("%i", &numero);

    if(numero == 0){

    printf("O numero digitado eh 0");

    }

    else if(numero == 2){

    printf("O numero digitado eh 2");

    }

    else{

    printf("O numero digitado nao eh 0 nem 2");

    }

    return 0;

    }
  • O código só imprime "O numero digitado eh 0" Se a variável numero tiver valor 0.
  • Usamos o else if quando queremos adicionar outra condição a ser verificada.
  • O código só entra no else caso nenhuma das condicionais anteriores for verdadeira.

    • Estruturas de repetição

  • Uma estrutura de repetição permite que uma sequência de comandos seja executada repetidamente, enquanto determinadas condições são satisfeitas.
  • Essas condições são representadas por expressões lógica (como, por exemplo, A>B; C==3; Letra == ‘a’)
  • Repetição com Teste no Início
  • Repetição com Teste no Final
  • Repetição Contada
  • O real poder dos computadores está na sua habilidade para repetir uma operação ou uma serie de operações muitas vezes.
  • Este repetição chamada laços (loop) é um dos conceitos básicos da programação estruturada
