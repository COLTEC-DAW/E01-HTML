Introdução

C é uma linguagem de programação. Normalmente é a primeira a ser aprendida por novatos, devido à sua simplicidade e seu poder, podendo funcionar como porta de entrada tanto para linguagens mais próximas da linguagem de máquina (Assembly) ou para linguagens de nível mais alto, incluindo Orientação a Objeto (Java, C++)

Básico do básico

Um programa em C segue a seguinte ordem:
-Chame as biblotecas que vai usar (EM TODOS OS NOSSOS EXEMPLOS USAREMOS A BIBLIOTECA BÁSICA "stdio.h")
-Escreva zero ou mais funções personalizadas
-Escreva a função int main(), onde o código vai estar. É possível usar void main, mas nós consideramos isso feio
-Lembre-se de terminar a int main() com return 0. Podemos retornar outras coisas, mas fique contente com o 0
-Comentários são indicados com // Eles não afetam o programa e servem para explicá-lo.
-O símbolo ; é usado ao fim de todas as linhas com comandos (a vasta maioria)
-As chaves ({ e }) servem para isolar partes de código, como loops, branchs e funções. Lembre-se de identar com (tab) para deixar o código visualmente organizado.

#include < stdio.h > //Aqui chamamos as bibliotecas

//Insira suas funções personalizadas aqui

int main(){
     //Insira seu código aqui
     return 0;
}

Entrada e saída

Use os comandos printf("saída") e scanf("%tipoentrada", &nomeentrada) para a entrada e saída de informações, especificamente.

//Esse código imprime Hello World na tela do terminal
#include < stdio.h >

int main (){
     printf("Hello World\n"); //O código \n indica um parágrafo
     return 0;
}

//Esse código imprime o valor digitado na tela do terminal
#include < stdio.h >

int main (){
     int entrada;
     scanf("%d\n", &entrada); //d é o código para inteiros na base 10
     printf("Você inseriu %d\n", entrada);
     return 0;
}

Variáveis e expressões matemáticas

Variáveis são espaços na memória do computador onde valores podem ser guardados. Variáveis tem nomes e podem representar vários tipos de informação. Os mais comuns são:
     int: Números inteiros
     float: Números reais
     char: Caracteres

Expressões matemáticas básicas usam parênteses e os símbolos de + - * e / para as operações básicas. % é usado para o resto de divisões. Há funções especiais para operações mais complicadas

int variavel1, variavel2, resultado;
variavel1 = 2;
variavel2 = 3;

resultado = (variavel1+variavel2)*variavel1;
printf("%d\n", resultado); //O valor mostrado será 10

resultado = variavel1+variavel2*variavel1;
printf("%d\n", resultado); //O valor mostrado será 8

resultado = (variavel2+5)/variavel1;
printf("%d\n", resultado); //O valor mostrado será 4

resultado = (variavel1+variavel2)%variavel1;
printf("%d\n", resultado); //O valor mostrado será 1

Estruturas condicionais

Estruturas condicionais permitem que o programa "faça escolhas", executando um ou outro trecho de código baseado em alguma condição. Observe os exemplos.

//A função numDivisores(a) retorna o número de divisores de um inteiro
// X é um inteiro positivo inserido pelo usuário

if (numDivisores(X) == 2){
    printf("%d é primo\n", X);
}


if (numDivisores(X) == 2){
    printf("%d é primo\n", X);
}
else{
    printf("%d não é primo\n", X);
}


if (numDivisores(X) == 2){
    printf("%d é primo\n", X);
}
else{
    printf("%d não é primo", X);
    if (numDivisores(X)%2 == 0){
        printf("O número de divisores é par\n");
    }
}

if (numDivisores(X) == 2){
    printf("%d é primo\n", X);
}
else if (numDivisores(X)%2 == 0){
    printf("O número de divisores é par\n", X);
}


//Tente encontrar a diferença entre os últimos dois exemplos


switch (numDivisores(X)){
    case 1: printf("O número tem um divisor\n");
    break;
    case 1: printf("O número tem um divisor\n");
    break;
    case 2: printf("O número tem dois divisores\n");
    break;
    default: printf("O número tem mais de dois divisores\n");
    break;
}

Estruturas de repetição

Estruturas de repetição permitem que o mesmo trecho de código seja excutado várias vezes. Execute os exmplos e tente entender as diferenças. //i é uma variável inteira

i=0;
while (i < 10){
    printf("%d\n", i);
    i++; // i = i+1
}



i=0;
do {
    printf("%d\n", i);
    i++;
} while (i < 10);



for (i=0; i < 10; i++){ //O for é mais complicado. A primeira parte é um comando executado antes de loop, a segunda uma condição que faz o loop rodar enquanto for verdadeira, a terceira uma ação repetida ao fim de cada loop
    printf("%d\n", i);
}

Como executar seu programa:

Executar é simples! Encontre o terminal do seu sistema UNIX e digite os seguintes comandos:

$ gcc nomeprograma.c
