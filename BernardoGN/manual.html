Manual de programação em C!

Introdução

C é uma linguagem de programação composta por funções e procedimentos que realiza uma série de série de comandos e manipulação de variáveis para conclusão de um objetivo.
A primeira função a ser executada chama-se main. Em programas mais simples, todos os comandos podem ser feitos no main, mas conforme a complexidade aumenta é viável a
utilização de outras funções e procedimentos. É necessário a utilização de ';' para indicar o encerramento de um comando. Ao final do programa, utiliza-se return 0, indicando que
a tudo ocorreu como o esperado e a função main retornou o valor zero.

Exemplo de código:

		
			
	int main(){  
		...comandos... 
		return 0; 
	}

Variáveis e expressões matemáticas

Variáveis em C são locais armazenados na memória que recebem um tipo de dado. Toda variável precisa de um nome e de um tipo. As váriveis podem ser do tipo int (inteiro)
float (ponto flutuante, para números não inteiros), char (caractere) entre outros. Muitas expressões matemáticas são utlizadas na linguagem C, utilizam-se:

  • '+' para soma;
  • '-' para subtração;
  • '*' para multiplicação;
  • '/' para divisão;
  • '=' para igual.

Exemplo de código para cálculo da média aritmética de três provas:

		
	int main(){
		int prova1, prova2, prova3, media;
	
		printf("Digite a nota da prova 1: ");
		scanf("%d", &prova1);
	
		printf("Digite a nota da prova 2: ");
		scanf("%d", &prova2);
	
		printf("Digite a nota da prova 3: ");
		scanf("%d", &prova3);
	
		media = (prova1+prova2+prova3)/3;
		printf("\nA média aritmética é %d", media);

		return 0;
	}

Estruturas condicionais

Estruturas condicionais são amplamente utilizadas em C. Elas são uzadas quando você precisa dar um tratamento diferente para situações diferentes. É indicada por meio de if, else, e else if.
Existe também o switch que realiza a mesma função. Utilizam-se:

  • "&&" com o sentido de "e"
  • "||" com o sentido de "ou"
  • "==" para comparar se são iguais
  • "!=" para comparar se são diferentes
  • ">" maior
  • "<" menor
  • ">=" maior ou igual
  • "<=" menor ou igual

Exemplo de programa usando if:


	int main() {
		int resposta;
		
		printf("Entre 1 e 5, qual é a nota da minha beleza?\n");
		scanf("%d", &resposta);

		if (resposta == 1 || resposta == 2){
			printf("Fidapeste\n");
		}else if (resposta == 3){
			printf("Errado\n");
		}else if (resposta == 4){
			printf("Quase lá!");
		}else if (resposta == 5){
			printf("Acertou!");
		}else{
			printf("Só pode entre 1 e 5");
		}

		return 0;
	}

Exemplo de programa usando switch:


	int main(){
		int resposta;
		
		printf("Entre 1 e 5, qual é a nota da minha beleza?\n");
		scanf("%d", &resposta);

		switch(resposta){
			case 1 or 2: 
				printf("Fidapeste");
				break;
			case 3:
				printf("Errado");
				break;
			case 4:
				printf("Quase lá!");
				break;
			case 5:
				printf("Acertou!");
				break;
			default:
				printf("Só pode entre 1 e 5");
		}

		return 0;
	}

Estruturas de repetição

Estruturas de repetição são utilizadas quando se necessita realizar um mesmo processo mais de uma vez, poupando do penoso trabalho de repetir o código diversas vezes. As principais
estrutras de repetição utilizadas são o while, o do e o for. Os comandos dentro dessa estrutura de repetição ficam em loop até que certa condição seja atingida.

Exemplo de usando while:


	int main() {
		int i = 0, numeroDeAlunos, nota, aprovados = 0;

		printf("Número de alunos: ");
		scanf("%d", &numeroDeAlunos);

		while(i < numeroDeAlunos){
			printf("\nDigite a nota do aluno %d\n", i);

Como executar um programa em C

A forma de executar um programa em C vai depender do sistema operacional que você estiver utilizando. É necessário possuir gcc, que vai compilar seu programa e apontar os erros

  • Linux ou Mac
    1. Abra o terminal no diretório do arquivo
    2. Digite gcc nome_do_seu_programa.c -o executavel
    3. Digite ./executavel

    Outra forma:

    1. Abra o terminal no diretório do arquivo
    2. Digite gcc nome_do_seu_arquivo.c
    3. Digite ./a.out
  • Windows
    1. Abra o cmd no diretório do arquivo
    2. Digite gcc nome_do_seu_arquivo.c -o executavel.exe
    3. Digite executavel.exe ou simplesmente executavel
