Linguagem C


Introdução

A linguagem C foi criada por Dennis Ritchie nos laboratórios da Bell Telephone em 1972.

C foi criada com um propósito: ser usada no desenvolvimento de uma nova versão do sistema operacional Unix. A primeira versão do Unix utilizava Assembly. Então podemos dizer que desde o principio C foi uma linguagem criada por programadores para programadores.

A linguagem C é considerada de propósito geral, ou seja é uma linguagem capaz de ser usada para praticamente qualquer tipo de projeto.

Variáveis e Expressões Matemáticas

Variáveis

Uma variável é uma possição de memória que pode ser identificada através de um nome.

Podem ter seu conteúdo alterado por um comando de atribuição. Após a atribuição mudam de valor.

int numerador; //declara a variavel numerador
numerador = 3; //atribuiu um valor a variavel

Expressões Matemáticas

É possível realizar operações matemáticas em C, da seguinte forma:

  • Adição: utiliza-se o operador +
    numerador = 1 + 2; //adicao
  • Subtração: utiliza-se o operador -
    numerador = 2 - 1; //subtracao
  • Multiplicação: utiliza-se o operador *
    numerador = 3 * 2; //multiplicacao
  • Divisão: utiliza-se o operador /
    numerador = 2 / 4; //divisao

Estruturas Condicionais

Na linguagem C, temos 4 tipos de estruturas condicionais:

  • if
    if (condicao){ bloco de operacoes }
  • else
    if (condicao){ bloco de operacoes }
    else { bloco de operacoes }
  • else if
    if (condicao){ bloco de operacoes }
    else if (condicao){ bloco de operacoes }
    else { bloco de operacoes }
  • switch
    switch (expressao integral) {
    case valor1: bloco de operacoes
    break;
    case valorn: bloco de operacoes
    break;
    }

Estruturas de Repetição

Na linguagem C, temos 3 tipos de estruturas de repetição:

  • while
    while (condicao){ bloco de operacoes }
  • do while
    do {
    bloco de operacoes
    } while (condicao);
  • for
    for(i = 0; i < 10; i++){ bloco de operacoes }

Executando um código em C

Para executar um programa em C, é preciso primeiro compilá-lo. Para isto, vá até a pasta onde se encontra o código e digite:

gcc nome_do_programa.c

Em seguida, basta digitar o seguinte:

./a.out
