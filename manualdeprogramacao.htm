Manual de programação

Introdução

A programação em C é muito simples e muito utilizada. A linguagem C foi uma das primeiras linguagens formadas e sua versão orientada á objetos, o C++, é uma das linguagens mais utilizadas atualmente.

É possível fazer diversas coisas em C, por exemplo, para fazer um "Hello world" use o código:

void main(){
  printf("Hello world");
}

Variáveis e expressões matemáticas

Também é possível fazer contas matemáticas, inclusive utilizando variáveis, por exemplo:

void main(){
  int x, y, res;
  scanf("%i %i", &x, &y);
  res = x + y;
  printf("O resultado é %i", res);
}

Estruturas condicionais

Caso você queira que uma função seja executada apenas se uma condição seja verdadeira você pode fazer assim:

if([condição]) função();

Estruturas de repetição

Agora, se você quiser que uma função se repita algumas vezes você pode inserir essa linha:

for(int i=0;i< [número de vezes];i++) funçao();
