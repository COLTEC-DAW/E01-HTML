Prova de Múltipla Escolha sobre HTML

  1. Um usuário de um computador com Windows 7 está utilizando o navegador Google Chrome, versão 65, para realizar a busca por informações sobre cursos de informática. Ao localizar uma página com informações interessantes, o usuário decide por salvar a página. Nessa situação, um dos tipos de salvamento da página é: Página da web
    1. Somente HTML
    2. Imagem JPEG
    3. Formato PDF
    4. Somente texto
    5. Formato impressão
  2. Assinale a alternativa INCORRETA:
    Trecho de HTML <html>
    <title> <b>
    </b> </title>
    </head>
    <body>
    </body>
    </html>
    1. O código HTML está incompleto.
    2. O Titulo será apresentado em negrito na tela
    3. A renderização irá depender de cada navegador
    4. A tag HEAD está incompleta
  3. Assinale a alternativa correta:
    1. HTML é um programa que favorece o desenvolvimento de arquivos CSS
    2. HTML e CSS são as tecnologias utilizadas na construção de páginas da Web: HTML (html e xhtml) para a estrutura, CSS para o estilo e layout, incluindo WebFonts
    3. Ambos são os conceitos fundamentais aplicados em dados abertos conectados
    4. CSS é uma linguagem de programação poderosa e requer HTML para seu funcionamento
  4. HTML é uma linguagem de marcação que descreve a estrutura de um documento. Essa linguagem é utilizada para desenvolver páginas Web. Sobre o HTML seguem-se três afirmações:
    I - Páginas criadas em HTML sempre são visualizadas da mesma forma em navegadores diferentes, mesmo quando executados em plataformas de hardware diferentes.
    II - As páginas escritas em HTML são arquivos de texto no formato ASCII.
    III - O HTML possui um conjunto previamente definido de tags, no entanto o usuário pode criar as suas próprias tags. É correto apenas o que se afirma em
    1. I
    2. II
    3. III
    4. I e II
    5. II e III
