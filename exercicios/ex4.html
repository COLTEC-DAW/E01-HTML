Tutorial da linguagem C!

Introdução


            #include<"stdio.h">
            int main() {
                printf("Hello World\n");
                return 0;
            }

#include inclui a biblioteca para, nesse caso, funções principais da linguagem como o printf. Esse comando é necessário para escrever algo na tela/terminal, logo utilizamos-lo para escrever o seu primeiro Hello World! O int main(){} é a função principal do seu programa, tudo o que precisa ser executado está no "main". O return é para indicar que acabou e foi tudo completado com sucesso. O significade de "0", na computação, é que foi executado sem nenhum erro, se tivesse algum erro o return seria diferente de 0

Variáveis e Expressões Matemáticas


            ...
            int a = 6, b = 5;
            printf("%d", a+b);
            printf("%d", a*b);
            printf("%d", a%b);
            printf("%d", a-b);
            ...

"int" é um tipo primitivo em C que significa inteiro, logo "a" e "b" são números inteiros com valores, respectivamente de 6 e 5. Na segunda linha é utilizado o uma expressão matemática de soma e escrevemos-la no terminal. O "%d", significa que estamos nos referenciando a um número inteiro. A segunda expressão é multiplicação, a quarta é a subtração. A terceira pode ser um pouco diferente do habitual que significa o resto, ou seja, dividirá a/b(6/5) e escreverá no terminal o resto dessa divisão que seria, no caso, "1".

Estruturas Condicionais


            if(4<3) {printf("a");}
            else if {printf("4 não é menor que 3");}

            ...
            (4<3) ? printf("a") : printf("4 não é menor que 3");
            ...

Se traduzirmos para o português fica bem claro as duas primeiras linhas, se a condição for verdadeira, acontecerá o printf("a"), senão acontecerá printf("4 não é menor que 3"). A segunda forma de escrever esse mesmo if/else if que ocupa 2 linhas chama-se if ternário, tem o mesmo próposito, acho o if ternário útil para esse caso, o qual, é muito simples de ser escrito. Se for um if complexo opte pelo if/else if
