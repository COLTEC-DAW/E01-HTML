Introdução

Olá. Aprenderemos um pouco de programação na linguagem C. Para isso, o
conteúdo que será passado aqui será dividido em 4 módulos, quais sejam:

  • Variáveis e Expressões Matemáticas;
  • Estruturas Condicionais;
  • Estruturas de Repetição;
  • Como Executar um Programa em C.

    • Variáveis e Expressões Matemáticas

    Para começar, é bom estar ciente de que toda variável precisa ser
    declarada antes de usada, e de que se usada ante de inicializada
    o resultado será impreciso, pois as operações serão feitas com o
    lixo na variável.

    int a;
    int b;
    int c;

    c = a+b; // c = valor imprevisivel

    d = e + f // error: uso de identificadores não declarados

    Os operadores aritméticos básicos em C são:

  • + para adição;
  • - para subtração;
  • * para multiplição;
  • / para divisão;
  • % para módulo (resto da divisão).

    • Genericamente, para efetuar qualquer uma dessas operações sobre as
    variáveis, basta representar a operação tal qual se faz normalmente
    no papel ou calculadora: colocando o símbolo correspondente entre as
    variáveis (atentando-se para a ordem de precedência de cada operador,
    que segeue a ordem comum da matemática). Ex: [a+(b-c)]/[d*e]

    Estruturas Condicionais

    Em c, exitem estruturas que só efetuam determinado trecho de código quando
    uma sentença é verdadeira, do contrário, efetuam outro trecho ou simplesmente
    ignoram ele.

    Uma dessas estruturas é o if o else e o else if.

    Se uma expressão for verdadeira,
    o if efetuará o bloco de código que ele contém, do contrário, o código continuará
    a ser executado a partir do bloco if.

    if(senteça){

    }

    O else executa um trecho de código quando a expressão dentro do if é falsa, ou seja,
    ele trabalha em conjunto com o if.

    if(senteça){

    }else {

    }

    O else if é uma junção do if e do else. Ele faz um segundo teste em outra expressão
    caso o if tenha retornado falso; basicamente atuando como um else que faz uma comparação.

    if(senteça){

    }else if(sentença){

    }

    Estruturas de Repetição

    As estruturas de repetição em C são:

    for, while e do while.

    O for possui três campos separados por ponto e vírgula que atuam
    no seu processo de decição. O primeiro campo serve para determinar o
    valor inicial de uma variável auxiliar. O segundo campo serve para constuir
    uma condição para a repetição. O terceiro campo serve para alterar o valor
    de uma variável, geralmente a auxiliar então definidia, para se alcançar a
    condição falsa para o encerramento do for.

    for(i = 0; i < n; i++ ){

    }

    O do while e o while são muito pareceidos. Ambos efetuam um teste para
    a repetição, com a diferença de que o do while efetua o trecho definido pelo
    while pelo menos uma vez, ainda que a sentença testada seja falsa.

    do{

    }while()

    while(){

    }

    Como Executar um Programa em C

    Para executar um código em C é necessário compilá-lo para que seja gerado o executável,
    após isso, basta executar o programa.
